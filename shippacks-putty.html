6.5. PuTTY Package¶

Allows to launch PuTTY sessions directly.

This plugin automatically detects the installed version of the official PuTTY distribution and lists its configured sessions so they can be launched directly from Keypirinha.

The PortableApps.com distribution of PuTTY can also be registered in package’s dedicated configuration file. Edit the configuration file for more info (search for the Configure Package: PuTTY item to do that).

6.5.1. Catalog Items¶

This package populates the Catalog with the following items:

  • Search for the items prefixed with PuTTY (by default). Their appear only if PuTTY is detected.
