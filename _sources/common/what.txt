|project| is a launcher for keyboard ninjas on Windows.

**Features:**

* Quickly find and launch plethora of items like applications, files,
  bookmarks, URLs, sessions (PuTTY, WinSCP, FileZilla), ...
* Evaluate a mathematical expression
* Convert numbers
* Switch to a running application searched by name
* Associate a keyword to a specific item
* ...
* An embedded Python3_ API

**Properties:**

* Lightning fast
* Fairly small memory footprint
* Battery friendly
* Extensible by writing :term:`plugin` scripts in Python3_.

You can see |project| as an alternative to Launchy and a cousin to Alfred.

See the :doc:`first` chapter for more information.
