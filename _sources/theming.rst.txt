=======
Theming
=======

Since v2.14, |project| GUI can be customized via the ``theme`` setting. Here are
the screenshots of some sample themes available by default.

Check out the default configuration file for (search for the
``[theme/...]`` sections).

.. tip::
    You can change the ``theme`` setting at runtime to visualize the new theme
    while Keypirinha's LaunchBox is displayed


Layout Themes
=============

TinyLayout
----------

.. image:: images/theme-layout-tiny.png
    :align: center
