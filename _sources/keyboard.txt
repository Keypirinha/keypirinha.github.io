==================
Keyboard Shortcuts
==================

LaunchBox
=========

+-----------------------------------------+------------------------------------+
| Action(s)                               | Shortcut(s)                        |
+=========================================+====================================+
| Show/Hide the LaunchBox                 | :kbd:`Ctrl+Win+K` (default)        |
+-----------------------------------------+------------------------------------+
| Next step (add args, choose action)     | :kbd:`Tab` or :kbd:`Alt+Right` (or |
|                                         | :kbd:`Space` while being |br| at   |
|                                         | first step and if the              |
|                                         | ``space_as_tab`` option |br| is    |
|                                         | enabled)                           |
+-----------------------------------------+------------------------------------+
| Forcefully go back to previous search   | :kbd:`Alt+Left`                    |
| step                                    |                                    |
+-----------------------------------------+------------------------------------+
| Force insert a space character while at | :kbd:`Ctrl+Space` or               |
| first step                              | :kbd:`Shift+Space`                 |
+-----------------------------------------+------------------------------------+
| Directly select an action               | :kbd:`Ctrl+Enter`                  |
+-----------------------------------------+------------------------------------+
| Show selected item's properties         | :kbd:`Alt+Enter`                   |
+-----------------------------------------+------------------------------------+
| Show Shell's "Properties" dialog for    | :kbd:`Shift+Alt+Enter`             |
| the selected **FILE** item              |                                    |
+-----------------------------------------+------------------------------------+
| Maximize the LaunchBox (toggle)         | :kbd:`Alt+X` or                    |
|                                         | :kbd:`Win+Up`                      |
+-----------------------------------------+------------------------------------+
| * Reset your current search (once)      | :kbd:`Esc`                         |
| * Hide the LaunchBox (twice)            |                                    |
+-----------------------------------------+------------------------------------+
| Hide the LaunchBox                      | :kbd:`Shift+Esc` or :kbd:`Alt+F4`  |
+-----------------------------------------+------------------------------------+
| Erase a single item from history        | :kbd:`Ctrl+Del`                    |
+-----------------------------------------+------------------------------------+
| Erase previous word                     | :kbd:`Ctrl+BackSpace`              |
+-----------------------------------------+------------------------------------+
| Show history items                      | :kbd:`Ctrl+Down` (or :kbd:`Down`   |
|                                         | if search state is clean)          |
+-----------------------------------------+------------------------------------+


Console
=======

+-----------------------------------------+------------------------------------+
| Action                                  | Shortcut(s)                        |
+=========================================+====================================+
| Hide the Console                        | :kbd:`Esc` or :kbd:`Alt+F4`        |
+-----------------------------------------+------------------------------------+
| Switch focus between command and log    | :kbd:`Ctrl+Tab`                    |
| controls                                |                                    |
+-----------------------------------------+------------------------------------+
| Validate command                        | :kbd:`Enter`                       |
+-----------------------------------------+------------------------------------+
| Erase previous word                     | :kbd:`Ctrl+BackSpace`              |
+-----------------------------------------+------------------------------------+
