===========
Env Package
===========

Available since version 2.13

A plugin to search your environment variables and copy their name and/or value.


Usage
=====

This plugin inserts the ``Env: Search`` item into the catalog, which accept a
search string as an argument to match variables names and values.

The following actions are availables, once you have found and selected the
desired variable:

* ``Copy``, to copy the full variable declaration expression ``NAME=value``
* ``Copy Value``
* ``Copy Name``
