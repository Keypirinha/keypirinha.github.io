==================
Keyboard Shortcuts
==================

LaunchBox
=========

+------------------------------------+-----------------------------------------+
| Shortcut(s)                        | Action(s)                               |
+====================================+=========================================+
| :kbd:`Ctrl+Win+K` (default)        | Show/Hide the LaunchBox                 |
+------------------------------------+-----------------------------------------+
| * :kbd:`Tab`                       | Next step (add args, choose action)     |
| * :kbd:`Alt+Right`                 |                                         |
| * :kbd:`Space` (while being at     |                                         |
|   first |br| step and if the       |                                         |
|   ``space_as_tab`` |br| option is  |                                         |
|   enabled)                         |                                         |
+------------------------------------+-----------------------------------------+
| :kbd:`Alt+Left`                    | Forcefully go back to previous search   |
|                                    | step                                    |
+------------------------------------+-----------------------------------------+
| * :kbd:`Ctrl+Space`                | Force insert a space character while at |
| * :kbd:`Shift+Space`               | first step                              |
+------------------------------------+-----------------------------------------+
| :kbd:`Ctrl+Enter`                  | Directly select an action               |
+------------------------------------+-----------------------------------------+
| :kbd:`Shift+Enter`                 | Execute the item without closing the    |
|                                    | LaunchBox, and reset                    |
+------------------------------------+-----------------------------------------+
| :kbd:`Ctrl+Shift+Enter`            | Execute the item without closing the    |
|                                    | LaunchBox, and go back to |br| the      |
|                                    | initial step of the current search      |
+------------------------------------+-----------------------------------------+
| :kbd:`Alt+Enter`                   | Show selected item's properties         |
+------------------------------------+-----------------------------------------+
| :kbd:`Shift+Alt+Enter`             | Show Shell's "Properties" dialog for    |
|                                    | the selected **FILE** item              |
+------------------------------------+-----------------------------------------+
| * :kbd:`Alt+X`                     | Maximize the LaunchBox (toggle)         |
| * :kbd:`Win+Up`                    |                                         |
+------------------------------------+-----------------------------------------+
| :kbd:`Esc`                         | * Reset your current search (once)      |
|                                    | * Hide the LaunchBox (twice)            |
|                                    |                                         |
|                                    | Also, see the ``escape_always_closes``  |
|                                    | setting.                                |
+------------------------------------+-----------------------------------------+
| * :kbd:`Shift+Esc`                 | Hide the LaunchBox                      |
| * :kbd:`Alt+F4`                    |                                         |
+------------------------------------+-----------------------------------------+
| :kbd:`Ctrl+Del`                    | Erase a single item from history        |
+------------------------------------+-----------------------------------------+
| :kbd:`Ctrl+BackSpace`              | Erase previous word                     |
+------------------------------------+-----------------------------------------+
| * :kbd:`Ctrl+Down`                 | Show history items                      |
| * :kbd:`Down`                      |                                         |
|   (if search state is clean)       |                                         |
+------------------------------------+-----------------------------------------+
| :kbd:`Ctrl+Home`                   | Scroll up to the first result           |
+------------------------------------+-----------------------------------------+
| :kbd:`Ctrl+End`                    | Scroll down to the last result          |
+------------------------------------+-----------------------------------------+
| :kbd:`F2`                          | Open Console                            |
+------------------------------------+-----------------------------------------+
| :kbd:`F3`                          | Open the Diagnostic Window              |
+------------------------------------+-----------------------------------------+
| :kbd:`F5`                          | Refresh Catalog |br| Identical to       |
|                                    | clicking the ``Refresh Catalog`` menu   |
|                                    | or launching the |br|                   |
|                                    | ``Keypirinha: Refresh Catalog`` item    |
+------------------------------------+-----------------------------------------+


Console
=======

+-----------------------+------------------------------------------------------+
| Shortcut(s)           | Action                                               |
+=======================+======================================================+
| :kbd:`Ctrl+Tab`       | Switch focus between command and log controls        |
+-----------------------+------------------------------------------------------+
| :kbd:`Enter`          | Validate command                                     |
+-----------------------+------------------------------------------------------+
| :kbd:`Ctrl+BackSpace` | Erase previous word                                  |
+-----------------------+------------------------------------------------------+
| * :kbd:`Esc`          | Close the Console                                    |
| * :kbd:`Alt+F4`       |                                                      |
+-----------------------+------------------------------------------------------+


Diagnostic Windows
==================

+-----------------------+------------------------------------------------------+
| Shortcut(s)           | Action                                               |
+=======================+======================================================+
| :kbd:`F5`             | Refresh (full content)                               |
+-----------------------+------------------------------------------------------+
| :kbd:`F6`             | Refresh then Copy privacy-safe content to |br|       |
|                       | clipboard so it can be pasted online to report |br|  |
|                       | an issue for example.                                |
+-----------------------+------------------------------------------------------+
| * :kbd:`Esc`          | Close the window                                     |
| * :kbd:`Alt+F4`       |                                                      |
+-----------------------+------------------------------------------------------+
