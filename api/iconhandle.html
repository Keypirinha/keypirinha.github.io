11.2.5. IconHandle class¶

class keypirinha.IconHandle(icon_id)¶

An opaque handle object created by Plugin.load_icon() and by load_icon().

free()¶

Frees the icon associated with this handle.

Returns:
A boolean value to indicate whether resources have actually
been freed or not. This method may return False if it has been called already for this handle, or by an other IconHandle object pointing to the same icon resource. You usually do not need to check the return value.
Return type:bool

Warning

This method must be called explicitly. It will not be called by the destructor of this object.

is_init()¶

Returns a boolean to indicate whether this handle has been free() or not.
