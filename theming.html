8. Theming¶

Since v2.14, Keypirinha GUI can be customized via the theme setting. Here are the screenshots of some sample themes available by default.

Check out the default configuration file for (search for the [theme/...] sections).

Tip

You can change the theme setting at runtime to visualize the new theme while Keypirinha’s LaunchBox is displayed

8.1. Layout Themes¶

8.1.1. TinyLayout¶

[gui]
theme = TinyLayout

8.1.2. MediumLayout¶

[gui]
theme = MediumLayout

8.1.3. JumboLayout¶

[gui]
theme = JumboLayout

8.1.4. Classic Theme (pre v2.14)¶

[gui]
theme = AnthraciteFull

8.2. Color Themes¶

8.2.1. Default Colors¶

[gui]
theme = MediumLayout, SolarizedLightColors

8.2.2. SolarizedLightColors¶

[gui]
theme = MediumLayout, SolarizedLightColors

8.2.3. SolarizedDarkColors¶

[gui]

8.2.4. ElectricStateColors¶

[gui]

[gui]
