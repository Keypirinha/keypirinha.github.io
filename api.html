7. Extending Keypirinha (API)Â¶

Keypirinha offers a flexible way to be extended by embedding a Python3 interpreter and exposing some of its core features via an API and two Python modules named keypirinha and keypirinha_util.

Before reading further, you may want to get familiar with the Packages section to picture how Keypirinha loads and interprets its packages and their plugins, as well as the subtleties between the different types of packages.
