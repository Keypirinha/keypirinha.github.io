4. Keyboard Shortcuts¶

4.1. LaunchBox¶

Action(s) Shortcut(s)
Show/Hide the LaunchBox Ctrl+Win+K (default)
Next step (add args, choose action) Tab (or Space while being at first
step and if the space_as_tab
option is enabled)
Force insert a space character while at first step Ctrl+Space or Shift+Space
Directly select an action Ctrl+Enter
Show selected item’s properties Alt+Enter
Show Shell’s “Properties” dialog for the selected FILE item Shift+Alt+Enter
Maximize the LaunchBox (toggle) Alt+X or Win+Up
  • Reset your current search (once)
  • Hide the LaunchBox (twice)
 Esc
Hide the LaunchBox Shift+Esc or Alt+F4
Erase a single item from history Ctrl+Del

4.2. Console¶

Action Shortcut(s)
Hide the Console Esc or Alt+F4
Switch focus between command and log controls Ctrl+Tab
Validate command Enter
