2. First steps¶

The first thing to know about Keypirinha is that you can use almost every of its features by using your keyboard only. Keep that in mind!

2.1. Start Keypirinha¶

By default, Keypirinha does not require any special configuration before being launched for the first time. It is a resident application, which means it directly goes to the background when it is started. You can ensure Keypirinha is running by checking for its icon in the system tray’s notification area. It is normally located next to the system clock on your desktop, in the lower-right corner of your main screen.

2.2. Open the LaunchBox¶

Once Keypirinha is running, press Ctrl+Win+K (by default) to display the LaunchBox...

2.3. First Search¶

You can now search for the task you wish to execute. Say you want to launch an application, Firefox for example. Start typing its name and it will soon show up in the results list as long as Firefox is installed on your machine.

Once the desired item is selected (you can use the Down, Up, PgDown and PgUp keys for that), press Enter to launch it. Keypirinha will then automatically go back to background...

2.4. Second Search¶

Keypirinha remembers the tasks you execute the most so if you want to launch an already executed task, chances are high that you will only need to type the very first couple of letters of its name to get the task at the top of the results list.

Notice how the score of the Firefox item is much higher now.

Note

The score numbers you observe on your machine might differ from the one displayed on that screenshot.

2.5. Leave a Search¶

May you wish to stop what you were doing on the LaunchBox, press the Esc key. Once, to reset your search. Twice, to close the dialog. One time might be enough if you didn’t type anything.

Tip

Standard Windows shortcuts are also implemented so the Alt+F4 combination will close the dialog as expected.

2.6. Configuration¶

See the Configuration section for more info.

2.7. Extending Keypirinha¶

Despite its apparent simplicity, Keypirinha is a generic tool that is able to welcome a very wide range of features not offered yet by the official bundled plugins. An API allows to extend Keypirinha‘s features to make them fit your needs.

See the Extending Keypirinha (API) section for more info.
